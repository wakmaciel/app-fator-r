# 📊 Fator R - Web App

Uma aplicação web (PWA) responsiva e otimizada para dispositivos móveis desenvolvida para auxiliar no cálculo, acompanhamento e planejamento do **Fator R**. O aplicativo permite monitorar o faturamento, a folha de pagamento e analisar cenários tributários diretamente no smartphone ou navegador.

## 📱 Funcionalidades

- **Cálculo Automatizado do Fator R**: Descubra instantaneamente se a empresa se enquadra no Anexo III ou Anexo IV do Simples Nacional.
- **Gráficos Interativos**: Visualização clara da evolução do faturamento e da folha de pagamento utilizando Chart.js.
- **Lançamentos Mensais**: Histórico de dados de faturamento e pró-labore/salários.
- **Gestão de Empréstimos**: Acompanhamento de contratos, parcelas pagas e saldo devedor.
- **Instalável (PWA)**: Pronto para ser adicionado à tela inicial do iOS (Safari) e Android (Chrome) funcionando como um aplicativo nativo.
- **Privacidade**: Todos os dados são persistidos localmente no navegador (LocalStorage), garantindo que nenhuma informação sensível seja enviada para servidores externos.

## ⚙️ Tecnologias Utilizadas

- **HTML5 & CSS3**: Estrutura robusta e estilização moderna e imersiva (Modo Escuro / Cyberpunk moderno).
- **JavaScript (Vanilla)**: Lógica de negócio, roteamento dinâmico e persistência de dados.
- **Chart.js**: Renderização de gráficos de performance.
- **Space Grotesk & Inter**: Fontes tipográficas modernas importadas via Google Fonts.

## 📲 Como Instalar no iOS (iPhone / iPad)

Como este projeto está configurado como um **Progressive Web App (PWA)**, você pode instalá-lo sem passar pela App Store:

1. Abra o link do projeto no navegador **Safari**.
2. Toque no botão de **Compartilhar** (ícone do quadrado com uma seta para cima na barra inferior).
3. Role a lista de opções para baixo e selecione **"Adicionar à Tela de Início"**.
4. Confirme o nome do aplicativo e toque em **Adicionar**.
5. O ícone do **Fator R** aparecerá na sua tela inicial e abrirá em tela cheia (sem as barras do navegador).

## 🚀 Como Executar Localmente

Se quiser rodar o projeto no seu computador:

1. Faça o clone deste repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` diretamente em qualquer navegador moderno.
